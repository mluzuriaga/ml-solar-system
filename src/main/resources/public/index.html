Sistema Solar - Mercado Pago

El sistema de pronóstico para Betasoides, Vulcanos y Ferengis.

Uso:

Importante: El cloud free utilizado tiene limite de request por hora a la BD, por lo tanto el cron corre una vez cada 15 minutos.

Método: GET
Ruta: http://ml-solar-system.herokuapp.com/solar-system/clima?dia=x
Descripción: Obtiene el pronóstico para el día 'x'. El número 'x' debe ser la cantidad de dias desde la fecha inicial del ultimo reporte generado.

Método: GET
Ruta: http://ml-solar-system.herokuapp.com/solar-system/reporte
Descripción: Obtiene el ultimo reporte generado.

Método: POST
Ruta: http://ml-solar-system.herokuapp.com/solar-system/nuevo?dias=x
Descripción: Genera un nuevo pronóstico a 'x' dias partir de la fecha final del último reporte generado.

Método: POST
Ruta: http://ml-solar-system.herokuapp.com/solar-system/parar
Descripción: Stop del proceso que calcula el pronostico.
